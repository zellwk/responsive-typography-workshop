Wonky whitespace

Vertical Rhythm

  • Set spaces between elements to a multiple of the baseline
  • Set leading of text elements to a multiple of the baseline

Vertical Rhythm

html {
	font-size: 16px;
	line-height: 1.4; // 22.4px
}


h1 {
	font-size: 36px;
	line-height: 44.8px; // 22.4px * 2 // Note: Line-height is ≈ 1.25.
	margin: 22.4px;
}

Change font-sizes

5 min

Change the font-size of your body text

  • All elements should scale
  • Look at the site from your mobile too!

Change font-sizes

h1 { font-size: 37.9px; }

@media (min-width: 600px) {
	h1 { font-size: 47.4px }
}

p { font-size: 16px; }

@media (min-width: 600px) {
	p { font-size: 20px; }
}

Change font-sizes

html { font-size: 16px; }

@media (min-width: 600px) {
	html { font-size: 20px; }
}

h1 { font-size: 37.9px; } /* 16 * 1.333 * 1.333 * 1.333 */

@media (min-width: 600px) {
	h1 { font-size: 47.4px } /* 20 * 1.333 * 1.333 * 1.333 */
}

Use relative units

html { font-size: 16px; }

@media (min-width: 600px) {
	html { font-size: 20px; }
}

h1 {
  font-size: 2.369em; /* 1.333 * 1.333 * 1.333 */
}

Use relative units

  • Em (em)
  • Rem (rem)
  • Percentages (%)
  • Viewport units (vw)

The EM unit

  • 1em = current font-size
h1 {
  font-size: 20px; /* 1em = 20px */
  margin: 1em; /* 20px */
}

 

 

The EM unit

  • 1em = current font-size
h1 {
  font-size: 1em
}
  • It depends!

 

 

The EM unit

  • 1em = current-font-size || inherited-font-size
html {
  font-size: 16px;
}

h1 {
  font-size: 1em; /* 16px */
}

The EM unit

  • What's the value of these?
p {
	font-size: 1em; // 16px
	margin: 1em; // 16px
}

h1 {
  font-size: 2em; // 32px 
  margin: 1em; // 32px 
}

Responsive MS

15 min

Responsive MS

  1. Use a smaller ratio smaller
  2. Add a base
  3. Add a ratio
  4. Change the ratio at different breakpoints

Use a smaller ratio

html {
	font-size: 16px;
	@media (min-width: 37.5em) {
	  font-size: 20px
	}
}

h1 {
	font-size: 1.95em; // 1.253
	@media (min-width: 37.5em) {
	  font-size: 2.44em; // 1.254
	}
}

Add a base

html {
	font-size: 16px;
	@media (min-width: 37.5em) {
	  font-size: 20px
	}
}

h1 {
	font-size: 1.618em; // step 3
	@media (min-width: 37.5em) {
	  font-size: 2.25em; // step 4
	}
}

Add a ratio

html {
	font-size: 16px;
	@media (min-width: 37.5em) {
	  font-size: 20px
	}
}

h1 {
	font-size: 1.563em; // step 3
	@media (min-width: 37.5em) {
	  font-size: 1.953em; // step 4
	}
}

Responsive MS

@import 'path-to-modular-scale/modular-scale';

$ms-base: 1em; // default
$ms-ratio: 1.618; // default

h1 {
	font-size: ms(3);
	@media (min-width: 37.5em) {
		font-size: ms(4);
	}
}
